# Backend

## Desenvolver uma API de produtos com os seguintes campos: id, nome, data de cadastro
### Requisitos
  - Java 14
  - Spring Boot
  - Spring Data
  - Swagger
  - Postgresql
  
## Criar API com os seguintes endpoints:
  - Listagem de produtos
  - Cadastro de produto
  - Edição de produto
  - Informações do produto (Detalhes)
